# Web Console

Minimal React + Vite shell for the TikTok automation backend. Includes React Query data layer, route guard placeholder, and sample pages for health + job creation.

## Commands

```bash
npm install
npm run dev
npm run test     # unit + a11y
npm run test:e2e # start dev server and run Playwright
```

Set `VITE_API_BASE` to point at the backend (defaults to `http://localhost:3000`).
